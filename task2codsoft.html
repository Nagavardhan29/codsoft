Projects

Project 1:IoT-Based Anti Theft Flooring System

The IoT-Based Anti-Theft Flooring System is a cutting-edge security solution designed to protect valuable assets. It uses embedded smart sensors to detect unusual movements or pressure changes on the floor. When unauthorized access is detected, real-time alerts are sent to the owner's smartphone via a connected mobile app. The system can also trigger alarms and activate cameras to capture evidence. This advanced technology ensures the safety of high-value items in various settings, such as homes, offices, and museums. It provides a seamless and effective security measure through the power of IoT.

Project 2:To-Do List Application

The To-Do List application is a web-based tool designed to help users manage their tasks efficiently. Users can add new tasks, edit existing ones, and delete completed or unwanted tasks. The application leverages local storage to save tasks, ensuring that the list persists even after the browser is closed. A clean, user-friendly interface makes task management simple and intuitive. The app is built using HTML, CSS, and JavaScript, showcasing responsive design for usability on various devices. This tool is ideal for anyone looking to organize their daily activities effectively.
